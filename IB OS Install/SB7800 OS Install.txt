NVIDIA Mellanox SB7800 InfiniBand Switch OS 재설치
Mellanox OS -> NVIDIA SB7800 OS
장애내용 : OS DISK I/O ERROR
기존 장착된 DISK : DEMSR-16GD07SC2DC-92
새로 장착한 DISK : 타무즈 GKX370 mSATA (64GB TLC)
iso 파일 마운트 : onie-recovery-x86_64-mlnx_x86-r0.iso
os 자동 설치 파일 : onyx-X86_64-3.10.4206.zip
---------------------------------------------------------------------------------------------------------------------------------------------
1. switch bios로 진입
bios 진입키 : Ctrl + B
id/pass : admin / admin
del = 부트메뉴
부트메뉴에서 마운트한 iso로 부팅
---------------------------------------------------------------------------------------------------------------------------------------------
2. grub 화면이 나오면 2번째꺼 눌러서 진입
##ONIE : Embed ONIE
재부팅되면 install os로 설치
##ONIE : Install OS
---------------------------------------------------------------------------------------------------------------------------------------------
3. onyx 파일이있는 usb 장착후 마운트
fdisk -l
---------------------------------------------------------------------------------------------------------------------------------------------
4. 실행 (애매함)
onie-nos-install /tmp/onxy-xxx.xx.bin
onie-nos-install onyx-X86_64-3.10.4206.zip
